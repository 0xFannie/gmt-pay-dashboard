# GMT Pay Dashboard - Tab结构设计

## 顶部区域（固定显示）
- 页面标题
- 侧边栏筛选器（日期、链、代币、卡面值）
- 核心指标卡片（5个关键指标）
- 水平分隔线

## Tab导航区域
7个Tab横向排列：

### Tab 1: 🌐 各链销售概览
内容：
- 各链卡片销量饼图
- 各链收入分布饼图
- 各链详细统计表（卡片数量、总面值、实际收入、手续费、平均费率）
- 销售时间趋势图（每日卡片销量和收入）
- 月度退款统计

### Tab 2: 💳 卡面值分析
内容：
- 各面值销售数量柱状图
- 各面值收入占比饼图
- 面值详细统计表

### Tab 3: 💰 代币使用分析  
内容：
- 各代币使用占比饼图
- 各链代币分布堆叠柱状图
- 代币详细统计表

### Tab 4: 💸 手续费分析
内容：
- 手续费率分布直方图
- 各链平均手续费率柱状图
- 手续费率说明（示例）
- 最低/最高/中位数手续费率指标

### Tab 5: 🎖️ NFT持有者折扣分析
内容：
- 总体统计（VIP用户数、购卡用户数、总卡片数、折扣享受率）
- 快照匹配情况饼图
- 折扣享受情况柱状图
- VIP购卡按链分布
- VIP购卡按面值分布
- Insights Summary（关键发现和业务洞察）

### Tab 6: 💵 卡片注销返还GGUSD分析
内容：
- 返还总览（总返还笔数、总返还金额、总返还卡片余额）
- 月度返还趋势图
- 返还明细表（时间、钱包地址、卡片余额、返还金额）
- 活动说明

### Tab 7: 📋 原始交易数据
内容：
- 完整交易记录表（时间、链、卡面值、实付金额、手续费、费率、代币、交易哈希）
- 数据导出功能

## 优势
✅ 页面不再需要上下滚动
✅ 用户可以快速切换到感兴趣的分析模块
✅ 每个Tab聚焦单一主题，信息更清晰
✅ 核心指标始终可见（在Tab上方）
